## Установка игры

Чтобы установить игру, выполните следующие шаги:

1. Создайте новую папку на вашем рабочем столе.
2. Скачайте файлы - 
3. Распакуйте скачанные архивы в созданную ранее папку.
4. Запустите исполняемый (`exe`) файл.
